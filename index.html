Scheme Interpreter
The Scheme Interpreter was a partner project from CS61A: The Structure and Interpretation of Computer Programs, completed in Fall 2016. To begin, simply hit the button. While it does not include all the features of the language, it supports the most commonly used ones and even tail recursion.

Examples of Inputs:
Creates a linked list of integers 1 and 2
scm> (cons 1 (cons 2 nil))
(1 2)

Creates a lambda function equivalent to adder = lambda x: lambda y: x + y in python
scm> (define (adder x) (define (add y) (+ x y)) add)
adder

scm> (define add-three (adder 3))
add-three

scm> (add-three 4)
7

Creates a function that recursively finds thte nth fibonacci number
scm> (define (fib n) (cond ((zero? n) 1) ((= n 1) 1) (else (+ (fib (- n 1)) (fib (- n 2))))))
fib

scm> (fib 1)
1

scm> (fib 3)
3

scm> (fib 10)
89
